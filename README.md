# Thari Tech Website

Official website for Thari Tech — CRM implementation and business automation solutions.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS v4

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deploy

Push to `main` — GitHub Actions builds and deploys automatically to GitHub Pages.

**Important:** After first push, go to:
Settings → Pages → Source → **GitHub Actions**

## Logo

Place your logo file as `public/logo.png` before running.
